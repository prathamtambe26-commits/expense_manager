# eXpense


## Setup
Install requirements and dependencies.
#### Windows
```
cd expense_manager/
python -m venv env
env/Scripts/activate.ps1
pip install -r requirements.txt
```
#### Linux
```
cd expense_manager/
python3 -m venv env
source env/bin/activate
pip install -r requirements.txt
```

## Run
```
python -m flask run
```
